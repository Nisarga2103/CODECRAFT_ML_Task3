# CODECRAFT ML Task 3

SVM Image Classification using Support Vector Machine.

This project implements a Support Vector Machine (SVM) model to perform image classification.  
The program trains the model and displays predicted labels and accuracy in the terminal.

## Steps to Run

1 Install dependencies

python -m pip install -r requirements.txt

2 Run the program

python svm_cat_dog_classifier.py

## Output

The program will display:

-Actual Labels  
-Predicted Labels  
-Accuracy  

Only a few results are displayed while the process continues for the remaining data.
